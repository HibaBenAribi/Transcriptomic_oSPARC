Transcriptomics raw data analysis pipeline.

  1. Check fastq files quality using the "FastQC" program.
  2. Trim and crop the fastq files using the "Trimmomatic" software.
  3. Map the reads to the reference genome using the "Star" software.
  4. Perform the expression quantification using the "featureCounts" software.
  5. Finally perform the differntial expression analysis using the "DESeq2" tool.
